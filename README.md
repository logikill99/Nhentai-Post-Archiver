# Nhentai-Post-Archiver
basically just downloads doujins marked English from the Nhentai API downloads them to the folder it is in and makes a list in a text document as to their tag numbers

(Can only check for and exclude one tag)

Written for python interpreter 3.9.0 and libraries for that version
THIS HAS NOT BEEN TESTED ON OTHER PYTHON VERSIONS AND MAY NOT WORK

SETUP PROCESS:

install the requests and hentai libraries via pip
commands:

pip install requests

pip install hentai

MAKE SURE YOU INSTALL THESE LIBRARIES FOR PYTHON 3.9.0

Hentai library must be on version 3.2.5
